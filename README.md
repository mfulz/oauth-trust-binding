## Structure

- draft/
  - draft-fulz-oauth-trust-binding-00.md   (Markdown Internet-Draft)
  - draft-fulz-oauth-trust-binding-00.xml  (xml2rfc skeleton, optional)
- docs/
  - developer_guide.md                     (Dev implementation guide, EN/DE)
  - regulatory_whitepaper.md               (Regulatory/DSGVO style doc, EN/DE)
- OUTREACH/
  - steps.md                               (1:1 checklist)
  - email_oauth_wg.txt
  - email_industry.txt
  - email_regulators.txt
  - email_media.txt

## How to use (short version)

1. Create a new GitHub repo, e.g. `oauth-trust-binding-extension`.
2. Drop the contents of this ZIP into that repo.
3. Submit `draft/draft-fulz-oauth-trust-binding-00.md` as an Internet-Draft via datatracker.ietf.org.
4. After you have the IETF link, open the OUTREACH/*.txt files and send them (replace `<IETF_LINK>` and `<GITHUB_LINK>`).
